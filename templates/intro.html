Welcome to the Cosine Calculator

This tool allows you to calculate the cosine of an angle using the Taylor series expansion. You can input angles in either degrees or radians.

How to Use

  • Select whether your input angle is in degrees or radians.
  • Enter your angle value. You can also use 'pi' in your input (e.g., 'pi', '2*pi').
  • Click the "Calculate" button to see the result.

Methods Used

The following methods are used in this program:

  • Factorial Function: Computes the factorial of a number, which is necessary for the Taylor series expansion.
  • Coterminal Angle Function: Adjusts the input angle to be within the range of 0 to 2π.
  • Taylor Series Expansion: Uses the Taylor series to approximate the cosine of the angle. The formula used is: 
    cos(x) = ∑(−1)^n * (x^(2n)) / (2n)!
    This series continues with the terms getting smaller, providing a more accurate approximation with more iterations.

Implementation Details

Here are the detailed steps of the implementation:

  • The program starts by importing the necessary libraries and defining the factorial function.
  • The coterminal function ensures the angle is within the standard range of 0 to 2π.
  • The Taylor series function calculates the cosine using a loop that iterates to improve accuracy.
  • The user interacts with the program by inputting an angle and choosing whether it is in degrees or radians.
  • The program handles errors and ensures the output is correctly signed based on the quadrant of the angle.
